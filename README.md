# 404notfound 
A HTML-based 404 not found HTML that can be used anywhere, regardless of what website do you have.\
Javascript, CSS and HTML are bundled onto 1 file.

To edit the site for your use, just download the file, then edit the title, and the link to your file. `index.html` is used for example.\
You can customize such as description, themes **(including dark mode)** and buttons.
